# 都内に特化した観光サイト

## サイト概要
都内の観光スポットについて知れる

### サイトテーマ
都内の観光地の情報が写真付きでわかる

### テーマを選んだ理由
コロナでなかなか遠出ができないからこそ、都内でお出かけできるスポットが最近話題だから。

### ターゲットユーザ
観光客はもちろん首都圏等に住んでいて都内へよく出かける方

### 主な利用シーン
都内に出かける際
